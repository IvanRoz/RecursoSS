Metodología

La materia se abordará a partir de la estrategia de aprendizaje orientada a proyectos y método heuristico, lo cual permitirá la utilización de las siguientes técnicas de aprendizaje:

  • Indagación documental
  • Lluvia de ideas
  • Discusión dirigida
  • Trabajo colaborativo
  • Práctica de laboratorio
  • Exposición de temas complementarios en equipo
  • Fichas de trabajo

Estrategias de aprendizaje

Proposito de la unidad de aprendizaje

  • Desarrollar aplicaciones de red, con base en interfaces de programación y arquitectura de comunicaciones, con alto nivel de calidad y eficiencia

Estrategias de aprendizaje utilizadas

  • Aprendizaje basado en problemas
  • Aprendizaje orientado a objetos

Evaluación

La primera evaluacion consiste de los siguientes puntos con sus respectivos valores:

  • Practicas - 60%
  • Tareas y ejercicios - 40%

La segunda evaluacion consiste de los siguientes puntos con sus respectivos valores:

  • Practicas - 40%
  • Tareas y ejercicios - 40%
  • Proyecto - 20%

La tercera y ultima evaluacion consiste de los siguientes puntos con sus respectivos valores:

  • Practicas - 40%
  • Tareas y ejercicios - 10%
  • Proyecto - 50%

Bibliografía

Beej’s Guide to Network Programming. Using Internet Sockets.

  • Autor: Brian “Beej Jorgensen” Hall.
  • Editorial: Publicado de manera independiente

Do-It-Yourself Multiplayer Java Games: An Introduction to Java Sockets and Internet-Based Games.

  • Autora: Annette M Godtland.
  • Editorial: Publicado de manera independiente

Hands-On Network Programming with C: Learn socket programming in C and write secure and optimized network code.

  • Autor: Lewis Van Winkle.
  • Editorial: Packt publishing

Java Network Programming: Developing Networked Applications.

  • Autor: Elliotte Rusty Harold.
  • Editorial: O'reilly media. Cuarta edición

Python Network Programming: Conquer all your networking challenges with the powerful Python language.

  • Autor: Abhishek Ratan, Eric Chou
  • Editorial: Packt publishing. Primera edición
